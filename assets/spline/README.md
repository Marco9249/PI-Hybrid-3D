# Local Spline Assets

Download your `.splinecode` files from Spline.design and save them here with the following names to make the project work offline:

1. `scene1.splinecode` -> (From <https://prod.spline.design/DqIqXA4OOQwaH4NN/scene.splinecode>)
2. `scene2.splinecode` -> (From <https://prod.spline.design/nmnEOKgSibIrAn4p/scene.splinecode>)

Once saved, the Spline Hub will load these files directly from your computer without needing an internet connection.
